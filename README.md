# Comm0ns Homebrew Tap

This repository contains the Homebrew formula for the Comm0ns CLI tools.

## Installation

```bash
brew tap Comm0ns-llc/comm0ns
brew install c0top
```

## Tools Available

### `c0top`
A blazing fast C++ CLI Dashboard for Comm0ns with built-in Discord OAuth.

## Updating

To update to the latest version, simply run:

```bash
brew upgrade c0top
```
